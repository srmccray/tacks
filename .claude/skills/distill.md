# /distill -- Summarize and Synthesize

Reduce a collection of findings into key insights.

## Usage

```
/distill <source>
```

## Process

1. **Input**: Read the gathered findings (from /gather output or specified source)
2. **Pattern**: Identify recurring themes, contradictions, or gaps
3. **Compress**: Reduce to 3-7 key insights with supporting evidence
4. **Output**: Prioritized insight list

## Notes

- Works on /gather output, code review findings, or any structured collection
- Pairs with /rank to prioritize the distilled insights
